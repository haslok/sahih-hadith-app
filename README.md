# صحيح - الأحاديث النبوية

بوت تليجرام وتطبيق ويب لعرض الأحاديث النبوية الصحيحة من الكتب الستة والأربعون النووية.

## المميزات

- 📚 الأحاديث من الكتب الستة + الأربعون النووية
- 🔍 البحث في الأحاديث
- 💾 حفظ المفضلة
- 🌙 ثيمة داكنة/فاتحة
- 📱 تطبيق PWA قابل للتثبيت
- 📖 عرض شرح كل حديث
- 📢 بث تلقائي للقناة والمستخدمين

## المتطلبات

- Node.js 18+
- حساب تليجرام وبوت

## التثبيت

```bash
npm install
```

## الإعداد

1. أنشئ ملف `.env` وأضف:
```env
BOT_TOKEN=توكن_البوت
DEVELOPER_ID=معرف_التليجرام_الخاص_بك
```

2. لتشغيل البوت:
```bash
npm run dev
```

## أوامر البوت

### أوامر المطور
- `/send_to_channel` - إرسال حديث للقناة
- `/send_periodic_to_channel` - إرسال حديث الفترة للقناة
- `/channel_msg <رسالة>` - إرسال رسالة مخصصة للقناة
- `/broadcast` - بث لجميع القنوات
- `/users` - عرض إحصائيات المستخدمين

### أوامر المستخدمين
- `/start` - بدء البوت

## البث التلقائي

- البث للقنوات: كل 5 ساعات
- البث للمستخدمين: كل 3 ساعات

## API

- `GET /api/random` - حديث عشوائي
- `GET /api/periodic` - حديث الفترة
- `GET /api/categories` - قائمة التصنيفات
- `GET /api/hadiths?bookId=` - أحاديث كتاب معين
- `GET /api/search?q=` - البحث
