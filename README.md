# صحيح - الأحاديث النبوية

تطبيق ويب لتعلم الأحاديث النبوية الصحيحة من الكتب الستة وأهل السنة والجماعة.

## المميزات

- 📚 الأحاديث من الكتب الستة + الأربعون النووية
- 🔍 البحث في الأحاديث
- 💾 حفظ المفضلة
- 🌙 ثيمة داكنة/فاتحة
- 📱 تطبيق PWA قابل للتثبيت
- 📖 عرض شرح كل حديث
- 🔎 البحث في جميع الكتب لمعرفة أي كتاب يحتوي الحديث
